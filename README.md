# my-blueside-ui

Элегантная Vue-библиотека компонентов в тёмной синей теме.  

---
  
— приглушённая синяя палитра,  
— мягкие градиенты,  
— чёткая типографика,  
— полная типизация и поддержка Vite + Vue 3.

Включает: `UiButton`, `UiInput`, `UiSelect`, `UiField` — всё, что нужно для лаконичного интерфейса.

Готова к использованию в продакшене. Подключай и создавай.
